# ScriptStack

A managed .NET scripting language to integrate into games (Unity3d, Godot) or any other application. See also 

- [Debugger](https://github.com/zarat/debugger) (for script debugging)
- [StackShell](https://github.com/zarat/stackshell) (simple console application)
- [GridRuler](https://github.com/zarat/GridRuler) (CSV analyzer that let you execute a script for each resulting element)
